
# 🛍️ Customer Shopping Behavior Analysis

## 📌 Project Overview

This project analyzes customer shopping behavior using 3,900 transactions across various product categories. The goal is to uncover insights into customer demographics, spending patterns, product preferences, and subscription behavior to support data-driven business decisions.

---

## 📊 Dataset Summary

* **Total Records:** 3,900
* **Features:** 18
* **Includes:**

  * Customer demographics (Age, Gender, Location, Subscription Status)
  * Purchase details (Item, Category, Amount, Season, Size, Color)
  * Behavior data (Discount, Purchase Frequency, Review Rating, Shipping Type)
* **Missing Values:** Review Rating column (handled during preprocessing)

---

## 🧹 Data Preprocessing (Python)

* Data cleaning using **pandas**
* Handled missing values using median imputation
* Standardized column names
* Feature engineering (age groups, purchase frequency)
* Removed redundant columns

---

## 🗄️ SQL Analysis (PostgreSQL)

Key business questions answered:

* Revenue by gender
* Subscribers vs non-subscribers comparison
* High-spending discount users
* Top-rated products
* Customer segmentation (New, Returning, Loyal)
* Top products per category
* Revenue by age group

---

## 📈 Dashboard (Power BI)

Created an interactive dashboard to visualize:

* Revenue trends
* Customer segments
* Product performance
* Shipping preferences

---

## 💡 Key Insights

* Subscribers contribute higher overall revenue
* Certain products heavily depend on discounts
* Loyal customers drive repeat purchases
* Specific age groups generate the most revenue

---

## 🚀 Business Recommendations

* Improve subscription offerings
* Introduce loyalty programs
* Optimize discount strategies
* Target high-value customer segments

---

## 🛠️ Tools & Technologies

* Python (pandas, NumPy)
